书签大师 隐私权政策

最后更新日期：2024年12月9日

数据收集

书签大师扩展程序不收集任何用户数据。所有操作均在用户本地设备上进行。

数据使用

书签数据

  • 仅通过 Chrome 浏览器的 bookmarks API 访问用户的书签
  • 所有书签数据仅存储在用户本地浏览器中
  • 不会将任何书签数据传输到任何外部服务器

存储数据

  • 使用 Chrome 的 storage API 存储用户偏好设置
  • 所有数据仅保存在用户本地设备上
  • 不会进行任何形式的数据上传

标签页信息

  • 仅用于"添加当前页面为书签"功能
  • 仅访问当前活动标签页的 URL 和标题
  • 不会记录或传输浏览历史

第三方服务

本扩展程序不使用任何第三方服务，也不与任何第三方共享数据。

数据安全

所有数据都存储在用户的本地设备上，受到 Chrome 浏览器的安全机制保护。
